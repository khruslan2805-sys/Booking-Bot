#Telegram Lead Bot

## Описание

Telegram-бот для сбора заявок от клиентов.

##  Функционал

* Ввод имени
* Кнопка отправки телефона
* Выбор услуги
* Отправка заявки администратору

##  Технологии

* Python 3
* aiogram 3
* FSM

##  Запуск

```bash
git clone <your_repo>
cd lead-bot

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt
```

Создать `.env`:

```
BOT_TOKEN=your_token
ADMIN_ID=your_id
```

Запуск:

```bash
python main.py
```

##  Применение

Подходит для:

* маркетинга
* строительных услуг
* агентств
* любых услуг
